## 注册常见问题

**注册信息能不能不真实？**

大部分信息可以用英文/拼音替代，不要求真实。但两样东西不能假：**邮箱**（要能收邮件）和**国家**（必须填 China，不能乱填其他国家）。填完以后提交，系统会发一封确认邮件，点一下邮件里的链接验证，账号就激活了。

**付款完成后多久开通？**

一般一到两分钟。最慢不超过半小时。开通后登录后台，找到你买的 VPS，点 **"KiwiVM Control Panel"** 就能看到 IP、SSH 端口和 root 密码。

**买完不满意能退款吗？**

可以。搬瓦工支持 30 天内申请退款，不满意就提工单，说明情况，一般会退回到原付款账户。

**优惠码只有首次有效吗？**

不是。**BWHCGLUKKB** 这个官方优惠码是循环有效的，续费时同样打折，不是只有第一次用。

**账号被砍单怎么回事？**

多半是注册时开着代理，或者 IP 归属地和填写国家不符。解决方法：重新注册一个账号，关掉所有代理，用本地真实网络操作，国家选 China，付款用支付宝。
